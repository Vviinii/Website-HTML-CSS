# VirtuHall 🏛️

A community engagement platform where people can connect, discuss local topics, and participate in decisions that affect their neighbourhood.

**Project by:** Azad Fakkusoglu & Vinayak Talawar — TH Aschaffenburg

## Features

- 🗣️ **Discussion topics** — Browse and engage with 6 local community topics
- 🖼️ **Photo gallery** — Community event photos with responsive grid layout
- 📬 **Contact section** — Direct contact link for questions and feedback
- 📱 **Fully responsive** — Works on mobile, tablet, and desktop
- ♿ **Accessible** — Keyboard-friendly navigation and semantic HTML

## Tech Stack

- HTML5 (semantic structure)
- CSS3 (custom properties, grid, flexbox, responsive design)
- Vanilla JavaScript (scroll-spy nav, mobile menu)
- Google Fonts (Playfair Display + Inter)

## Getting Started

Just open `index.html` in any modern browser — no build step required.

```bash
git clone https://github.com/your-username/virtuhall.git
cd virtuhall
open index.html
```
